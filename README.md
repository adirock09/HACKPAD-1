My custom hack pad with 5 Keys made for general shortcuts. It is based on SeedStudio XIAO RP2040.<br>

#  BOM<br>
1 x XIAO RP2040<br>
 4 x MX-Style switches<br>
 4 x Blank DSA keycaps<br>
 4 x M3x16mm screws<br>
 4 x M3x5mx4mm heatset inserts<br>
